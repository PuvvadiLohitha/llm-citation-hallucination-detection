# LLM Citation Hallucination Detection

This project focuses on detecting hallucinated academic citations generated by large language models (LLMs).

## Problem
Large language models sometimes generate fabricated references that appear realistic but do not exist in academic databases.

## Approach
This project proposes a multi-tier verification framework that includes:

1. Metadata validation using CrossRef API
2. Semantic similarity analysis using Sentence-BERT
3. Temporal consistency checks

## Technologies Used
Python  
Sentence Transformers  
Scikit-learn  
CrossRef API  
Semantic Scholar API  

## Results
The system achieved:

78% detection accuracy  
100% recall for hallucinated citations
